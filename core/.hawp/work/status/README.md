# Status Reports & Plan Files

Per-item plan files and status reports created during the intake loop.

Naming: `YYYY-MM-DD-<id>-<slug>.md` (e.g. `2026-04-30-task-003-kit-work-restructure-adr.md`).

Plan files use the template at `../../kit/templates/intake-plan.md`.
Each backlog item gets one plan file — see `../../kit/usage/INTAKE_WORKFLOW.md`.
